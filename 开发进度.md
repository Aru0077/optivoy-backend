# Optivoy Backend 开发进度（现行版本）

最后更新：2026-03-28

## 当前方向

项目已切换到“AI 行程生成”模式：

- 不再维护旧 `travel-routes` 线路模板模块
- Admin 仅维护点位：景点、购物、酒店
- 用户按城市选点并输入到达时间，由后端调用 AI 生成行程
- AI 必须返回严格 JSON，后端做结构审查后再返回前端

## Trip Planner API

| 方法 | 路径 | 鉴权 | 作用 |
|------|------|------|------|
| `GET` | `/trip-planner/cities` | 公开 | 查询可规划城市（景点/购物聚合）。 |
| `GET` | `/trip-planner/cities/:city/points` | 公开 | 查询该城市可选景点和购物点（不含酒店）。 |
| `POST` | `/trip-planner/generate` | 公开 | 提交到达时间+点位，调用 AI 生成严格 JSON 行程。 |
